# 戒

> 创意来自于戒游戏动机，以往都是直接卸载游戏，虽然现在上百G的游戏在千兆光纤与 SSD 下属于分钟级安装，但这个过程比较浪费硬盘寿命，而游戏念头往往产生于瞬间，只要阻断了这个过程即可降低欲望。

基于 Tauri 的应用程序“软禁用”，可以选择永久或定时禁用，后续如果后悔可主动恢复，但需要回答几个问题，类型包含恢复提示、强制等待、数学挑战、打字游戏…
